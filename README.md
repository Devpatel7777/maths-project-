Income Distribution Analysis
Overview

This project analyzes household income data using basic statistical methods and visualizations.
The goal is to understand income distribution, central tendency, and variability.

Dataset

The dataset contains 150 household records with the following features:

Household_ID
Age
Income
Education
Family_Size
Owns_House
Area (Urban/Rural)
Objectives
Calculate mean, median, and mode
Measure data spread using range, variance, and standard deviation
Analyze income distribution
Visualize data using plots
Tools Used
Python
Pandas
NumPy
Matplotlib
Seaborn
SciPy
Analysis Performed
1. Central Tendency
Mean: Average income
Median: Middle value
Mode: Most frequent value
2. Dispersion
Range: Difference between max and min
Variance: Spread of data
Standard Deviation: Square root of variance
3. Distribution
Histogram to visualize income distribution
Gaussian curve for normal distribution
4. Visualization
Boxplot (Area vs Income)
Scatter plot (Age vs Income)
Key Insights
Income data shows slight positive skew
Most households fall in medium income range
Few households have very high income
Indicates income inequality

How to Run
Download the dataset
Open the notebook in Jupyter or Google Colab
Run all cells step by step
Conclusion

This project demonstrates basic statistical analysis and visualization techniques to understand income distribution patterns in a dataset.
